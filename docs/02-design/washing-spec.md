## 1. 공정 개요

### 1.1 세척 공정 위치

세척 공정은 김치 생산 라인의 **첫 번째 공정**으로, 원재료 입고 검수 이후 절임 공정 이전에 위치합니다.

```
[원재료 입고]
     |
     v
[입고 검수 / QC 승인]
     |
     v
[세척 공정] <-- 본 화면 범위
     |
     v
[절임 공정] (LOT 연결)
     |
     v
[양념 배합]
     |
     v
[버무리기]
     |
     v
[포장 / 출고]
```

### 1.2 세척 공정 흐름도

```
원재료 반입
    |
    v
세척 배치 등록
(배치번호 자동 채번, 작업지시번호 연결, 원재료 종류/투입량 입력)
    |
    v
1차 세척 시작
(흐르는 물 또는 침지, 온도 확인)
    |
    v
[이물질 육안 검사]
    |--- 이물질 검출 ---> [이물질 검출 알림 발송] ---> [재세척 또는 폐기 처리]
    |
    v (이상 없음)
2차 / 3차 세척 (채소 종류별 기준 충족 시까지 반복)
    |
    v
세척 완료 처리
(완료 시간 기록, 세척 횟수 확정, 품질 등급 부여)
    |
    v
잔류 농약 / 위생 기준 확인 (HACCP CCP-W1)
    |--- 기준 초과 ---> [생산 중단, 품질관리자 알림, 로트 격리]
    |
    v (기준 충족)
절임 공정으로 LOT 이관
```

### 1.3 사용 설비 및 용수 관리

| 항목 | 내용 |
|------|------|
| 주요 설비 | 버블 세척기, 브러시 세척기, 고압 세척기, 세척조(스테인레스) |
| 용수 기준 | 먹는물관리법 기준 적합 수돗물 또는 지하수(정기 수질검사 필수) |
| 세척수 온도 | 채소류 5~15℃ (여름철 냉수 사용 권장, 배추 기준) |
| 세척수 교체 | 배치 교체 시 또는 오염 기준 초과 시 즉시 교체 |
| HACCP 관리점 | CCP-W1 (세척수 온도), CCP-W2 (이물질 제거 기준) |
| 일일 위생 점검 | 설비 청결 상태, 세척조 소독 여부, 용수 pH 확인 |

## 2. 사용자 스토리

### 2.1 생산관리자 관점

**US-WS-01: 오늘 세척 배치 현황 파악**
```
생산관리자로서,
오늘 진행 중인 세척 배치 전체 현황을 한눈에 보고 싶다.
이를 통해 생산 일정 지연 여부를 즉시 파악하고 조치할 수 있다.

인수 기준:
- 오늘 날짜 기준 세척 배치 목록이 대시보드 상단에 표시된다
- 배치별 상태(대기/진행중/완료/보류)가 색상 뱃지로 구분된다
- 각 배치의 투입량, 진행률, 경과 시간이 표시된다
- 작업지시번호를 클릭하면 해당 작업지시 상세 페이지로 이동한다
```

**US-WS-02: 세척 배치 등록**
```
생산관리자로서,
작업지시에 연결된 세척 배치를 등록하고 싶다.
이를 통해 세척 공정 착수를 MES에 정확히 기록할 수 있다.

인수 기준:
- 배치번호는 'WASH-YYYYMMDD-NNN' 형식으로 자동 채번된다
- 작업지시번호 드롭다운에서 오늘 대기 중인 작업지시 선택이 가능하다
- 원재료 종류 선택 시 해당 채소의 기본 세척 기준(횟수, 온도)이 자동 표시된다
- 등록 후 상태는 '대기'로 설정된다
```

**US-WS-03: 일별/월별 세척량 통계 조회**
```
생산관리자로서,
일별 및 월별 세척량 통계를 조회하고 싶다.
이를 통해 생산 능력 분석 및 계획 수립에 활용할 수 있다.

인수 기준:
- 일별 세척량 차트(bar chart)가 최근 30일 기준으로 표시된다
- 월별 세척량 합계 및 원재료 종류별 비율이 표시된다
- 기간 필터(시작일~종료일) 적용이 가능하다
- CSV 내보내기 기능이 제공된다
```

### 2.2 품질관리자 관점

**US-WS-04: 이물질 검출 알림 수신 및 처리**
```
품질관리자로서,
이물질 검출 알림을 즉시 수신하고 처리 지시를 내리고 싶다.
이를 통해 불량 원재료가 다음 공정으로 넘어가는 것을 막을 수 있다.

인수 기준:
- 이물질 검출 등록 시 품질관리자에게 시스템 내 알림이 발송된다
- 알림에는 배치번호, 원재료 종류, 검출 내용, 담당 작업자 정보가 포함된다
- 품질관리자는 '재세척 승인', '폐기 처리', '보류' 중 하나로 처리 지시를 등록한다
- 처리 지시 전까지 해당 배치는 '보류' 상태로 잠금 처리된다
```

**US-WS-05: HACCP 기준 충족 여부 확인**
```
품질관리자로서,
각 세척 배치의 HACCP 관리 기준점(CCP) 충족 여부를 확인하고 싶다.
이를 통해 식품 위생 규정 준수를 보장할 수 있다.

인수 기준:
- 세척수 온도가 기준 범위를 벗어나면 경고 표시가 붉은색으로 표시된다
- 세척 횟수가 채소 기준 미달 시 배치 완료 처리가 비활성화된다
- 각 배치의 CCP 충족 이력이 기록 및 조회 가능하다
- HACCP 기록부 양식으로 PDF 출력이 가능하다
```

**US-WS-06: 세척 품질 이력 조회**
```
품질관리자로서,
특정 LOT에 대한 세척 품질 이력을 소급 조회하고 싶다.
이를 통해 클레임 발생 시 원인 추적이 가능하다.

인수 기준:
- LOT번호 검색으로 해당 세척 배치 정보를 조회할 수 있다
- 세척 이력(온도, 횟수, 시간, 작업자, 품질등급)이 전부 조회된다
- 이물질 검출 이력 및 처리 결과가 포함된다
```

### 2.3 현장작업자 관점

**US-WS-07: 세척 배치 진행 처리**
```
현장작업자로서,
배정된 세척 배치를 시작하고 진행 상황을 입력하고 싶다.
이를 통해 내 작업을 정확히 기록하고 다음 공정에 인계할 수 있다.

인수 기준:
- '세척 시작' 버튼 클릭 시 시작 시간이 자동 기록된다
- 세척 횟수 증가 버튼(+1)으로 간편하게 횟수를 업데이트할 수 있다
- 세척수 온도는 직접 입력 또는 연동 센서 자동 입력이 가능하다
- '완료' 처리 시 품질 등급(A/B/C) 선택이 필수이다
```

**US-WS-08: 이물질 검출 등록**
```
현장작업자로서,
세척 중 이물질을 발견했을 때 즉시 시스템에 등록하고 싶다.
이를 통해 품질관리자에게 신속하게 보고하고 지시를 받을 수 있다.

인수 기준:
- 이물질 검출 버튼은 배치 상세 화면 내에서 항상 접근 가능하다
- 이물질 종류(돌/흙/해충/금속/기타), 발견 위치, 사진 첨부(선택)를 입력한다
- 등록 즉시 해당 배치는 '보류' 상태로 변경되고 품질관리자 알림이 발송된다
```

## 4. 화면 기능 목록

### 4.1 페이지 레이아웃 개요

```
/process/washing
├── PageHeader: "세척 공정 관리"
│   └── 부제목: "원재료 세척 배치 현황 및 품질 기록을 관리합니다."
│   └── breadcrumbs: [홈 > 공정관리 > 세척]
├── [탭 네비게이션]
│   ├── 탭1: 오늘 현황 대시보드
│   ├── 탭2: 세척 배치 관리
│   └── 탭3: 통계
└── [탭 컨텐츠 영역]
```

### 4.2 탭1 - 오늘 세척 배치 현황 대시보드

**목적**: 당일 세척 공정 전체 현황을 한눈에 파악

**구성 요소:**

| 컴포넌트 | 설명 |
|----------|------|
| KPI 요약 카드 (4개) | 오늘 총 세척량(kg) / 완료 배치 수 / 진행 중 배치 수 / 이물질 검출 건수 |
| 이물질 검출 알림 배너 | 미처리 이물질 검출 건이 있을 경우 붉은색 경고 배너 상단 노출 |
| 배치 현황 테이블 | 오늘 등록된 전체 배치 목록 (상태별 색상 뱃지 포함) |
| 원재료 종류별 세척량 도넛 차트 | 오늘 원재료 종류별 비율 시각화 |
| 실시간 갱신 | 30초 간격 자동 새로고침 (polling 방식) |

**배치 현황 테이블 컬럼:**
- 세척배치번호 / 작업지시번호 / 원재료 / 투입량(kg) / 세척횟수 / 수온(℃) / 상태 / 작업자 / 시작시간 / 경과시간 / 처리

**상태 뱃지 색상 규칙:**
- 대기(WAITING): 회색
- 진행중(IN_PROGRESS): 파란색
- 완료(COMPLETED): 초록색
- 보류(ON_HOLD): 주황색 (이물질 검출 등)
- 폐기(DISCARDED): 붉은색

### 4.3 탭2 - 세척 배치 등록/진행/완료 처리

**4.3.1 세척 배치 등록 모달/폼**

필드 목록:
- 작업지시번호: 드롭다운 (당일 대기 중 작업지시 목록)
- LOT번호: 작업지시 선택 시 자동 입력 (수정 가능)
- 원재료 종류: 드롭다운 선택 → 세척 기준 자동 표시
- 투입량(kg): 숫자 입력 (소수점 2자리)
- 세척수 온도(℃): 숫자 입력 (범위 유효성 검사 포함)
- 작업자: 현재 로그인 사용자 자동 입력 (변경 가능)
- 비고: 자유 텍스트

자동 표시 정보 (원재료 선택 시):
- 채소 종류별 최소/권장 세척 횟수
- 허용 수온 범위
- 세척 방법 안내
- 적용 HACCP 관리점

**4.3.2 배치 상세/진행 화면**

- 배치 기본 정보 표시 (읽기 전용)
- 세척 시작 버튼: 클릭 시 시작시간 자동 기록, 상태 IN_PROGRESS 변경
- 세척 횟수 입력: +1 버튼 또는 직접 입력
- 세척수 온도 갱신: 온도 재입력 가능 (이력 쌓임)
- 이물질 검출 등록 버튼: 항상 활성화 (진행 중 상태에서)
- 세척 완료 처리 버튼:
  - 최소 세척 횟수 미달 시 비활성화 + 경고 메시지 표시
  - 수온 범위 이탈 시 확인 팝업 표시
  - 완료 처리 시 품질 등급(A/B/C) 선택 필수
  - 완료 시간 자동 기록

**4.3.3 이물질 검출 등록 모달**

필드:
- 이물질 종류: 돌/흙/해충/금속/기타 선택
- 발견 위치: 텍스트 입력
- 상세 내용: 텍스트에리어
- 사진 첨부: 파일 업로드 (JPEG/PNG, 최대 5MB)

처리 결과:
- 등록 즉시 해당 배치 상태 ON_HOLD 변경
- 품질관리자 시스템 알림 발송
- 품질관리자가 '재세척 승인' 처리 시 상태 IN_PROGRESS 복귀

### 4.4 탭3 - 일별/월별 세척량 통계

**일별 통계 차트:**
- Bar Chart: 최근 30일 일별 세척량(kg)
- 원재료 종류별 색상 구분 (적층 막대 그래프)
- 일별 이물질 검출 건수 라인 표시 (보조 축)

**월별 통계 차트:**
- Bar Chart: 최근 12개월 월별 세척량
- 전월 대비 증감률 표시

**상세 집계 테이블:**
- 기간 필터(시작일~종료일)
- 원재료 종류별 필터
- 총 세척량, 평균 세척 소요 시간, 이물질 검출률, 평균 품질 등급 점수

**내보내기:**
- CSV 다운로드 버튼 (세척 배치 원본 데이터)
- HACCP 세척 기록부 PDF 출력

## 5. KPI 지표

| KPI | 정의 | 계산식 | 목표값 | 경고 기준 |
|-----|------|--------|--------|-----------|
| 일일 세척량(kg) | 당일 완료된 세척 배치의 총 투입량 합계 | SUM(input_weight_kg) WHERE status='COMPLETED' AND DATE(end_time)=오늘 | 생산 계획량의 100% | 생산 계획량의 80% 미만 시 경고 |
| 이물질 검출률(%) | 전체 배치 중 이물질 검출 배치 비율 | (이물질 검출 배치 수 / 전체 배치 수) × 100 | 1% 이하 | 3% 초과 시 경고 |
| 평균 세척 소요 시간(분) | 완료 배치의 평균 소요 시간 | AVG(wash_duration_min) WHERE status='COMPLETED' | 원재료 종류별 기준치 ±10% | 기준치 +20% 초과 시 경고 |
| HACCP CCP 준수율(%) | 수온/세척 횟수 기준 충족 배치 비율 | (CCP 기준 충족 배치 수 / 전체 완료 배치 수) × 100 | 100% | 99% 미만 시 즉시 경고 |
| 세척 품질 등급 A 비율(%) | 품질 등급 A 배치의 비율 | (등급 A 배치 수 / 완료 배치 수) × 100 | 80% 이상 | 60% 미만 시 경고 |

## 6. 비즈니스 규칙

### 6.1 채소 종류별 세척 기준 (HACCP 기준 반영)

| 원재료 | 최소 세척 횟수 | 권장 세척 횟수 | 세척수 온도 | 세척 방법 | HACCP 관리점 |
|--------|----------------|----------------|-------------|-----------|--------------|
| 배추 | 3회 | 3~4회 | 5~15℃ | 침지 + 브러시 | CCP-W1 |
| 무 | 2회 | 3회 | 5~15℃ | 고압 세척 + 침지 | CCP-W1 |
| 파(대파/쪽파) | 3회 | 3~4회 | 5~15℃ | 흐르는 물 + 침지 | CCP-W1 |
| 갓 | 3회 | 4회 | 5~12℃ | 침지 + 흐르는 물 | CCP-W1 |
| 기타 채소 | 2회 | 3회 | 5~15℃ | 흐르는 물 | CCP-W1 |

> 근거: 식품의약품안전처 「식품 및 식품첨가물공전」채소류 세척 기준 및 HACCP 관리 기준서

### 6.2 이물질 검출 시 처리 프로세스

```
이물질 검출 등록 (작업자)
    |
    v
배치 상태 → ON_HOLD (자동)
품질관리자 알림 발송 (자동)
    |
    v
품질관리자 현장 확인
    |
    +---> [경미한 이물질, 재세척으로 제거 가능]
    |         |
    |         v
    |     재세척 승인 → 배치 상태 IN_PROGRESS 복귀
    |     세척 횟수 초기화(또는 유지) → 재세척 진행
    |
    +---> [심각한 이물질 (금속/해충 등)]
    |         |
    |         v
    |     폐기 처리 → 배치 상태 DISCARDED
    |     폐기 중량 기록 → 생산관리자 통보
    |     원인 분석 보고서 작성 의무
    |
    +---> [추가 조사 필요]
              |
              v
          보류 유지 → 원인 규명 후 재판단
          품질관리자 메모 필수 입력
```

**이물질 등급별 처리 원칙:**
- 1등급 (금속, 해충): 즉시 폐기, 라인 점검 필수
- 2등급 (돌, 플라스틱): 폐기 원칙, 품질관리자 판단에 따라 재세척 가능
- 3등급 (흙, 잎 파편): 재세척 후 재검사

### 6.3 세척수 온도 기준 위반 처리

| 상황 | 시스템 동작 |
|------|-------------|
| 경고 범위 (기준±2℃) | 주황색 경고 표시, 작업 계속 가능 |
| 위반 범위 (기준±5℃ 초과) | 붉은색 경고 + 완료 처리 시 품질관리자 승인 필요 |
| 심각 위반 (0℃ 이하 또는 25℃ 이상) | 배치 일시 중단 권고, 품질관리자 즉시 알림 |

### 6.4 LOT 연결 규칙

- 세척 배치 완료 시 자동으로 절임 공정 대기 LOT 생성
- 동일 작업지시 내 복수의 세척 배치는 합산하여 하나의 절임 LOT로 연결 가능
- 이물질 검출로 폐기 처리된 배치의 LOT는 절임 공정으로 이관 불가

### 6.5 데이터 보존 규칙 (HACCP 기록 관련)

- 세척 배치 기록: 최소 3년 보존 (식품위생법 제40조 기준)
- 이물질 검출 이력: 최소 5년 보존
- 삭제 불가 (논리 삭제만 허용, 물리 삭제 금지)
