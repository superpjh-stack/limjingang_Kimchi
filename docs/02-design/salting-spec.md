## 1. 공정 개요

### 1.1 절임 공정의 위치

```
[1단계] 원재료 입고/검수
       ↓
[2단계] 세척 공정
       ↓
[3단계] 절임 공정  ← 본 페이지 대상
       ↓
[4단계] 탈수 / 2차 세척
       ↓
[5단계] 양념 공정
       ↓
[6단계] 포장 / 출하
```

### 1.2 절임 공정 단계별 흐름

```
[절임통 투입]
  - 세척 배치와 LOT 연계
  - 원재료(배추/무 등) 투입량 계량
  - 소금 용액 조배(소금 농도 목표값 설정)
        ↓
[소금 농도 초기 측정]
  - 소금 농도계(브릭스/%) 측정 및 기록
  - 허용 범위(목표 ±1%) 이탈 시 즉시 알림
        ↓
[중간 점검] (계획 시간의 50% 경과 시점)
  - 소금 농도 재측정
  - 절임 온도 확인
  - 원재료 상태 육안 점검
        ↓
[완료 판정]
  - 계획 절임 시간 도달 확인
  - 최종 소금 농도 측정 (HACCP CCP 기록)
  - 담당자 품질 체크 (합격/불합격 판정)
        ↓
[탈수 이관]
  - 합격 시: 탈수 공정으로 배치 이관
  - 불합격 시: 이상 처리 절차 진행
```

### 1.3 HACCP 핵심관리점(CCP) 관리

절임 공정은 HACCP 계획상 **CCP-2(절임)**로 지정됩니다.

| CCP 항목 | 한계기준(Critical Limit) | 모니터링 주기 | 담당자 |
|----------|--------------------------|---------------|--------|
| 소금 농도 | 10% ~ 15% (배추 기준) | 투입 직후, 중간, 완료 시 | 품질관리자 |
| 절임 시간 | 최소 4시간 ~ 최대 8시간 (계절 보정 적용) | 배치 시작/종료 기록 | 생산관리자 |
| 절임 온도 | 15℃ ~ 20℃ | 2시간마다 | 현장작업자 |

> HACCP CCP 일탈 발생 시 즉각 시정조치(Corrective Action) 기록이 법적으로 요구됩니다.

## 2. 사용자 스토리

### 2.1 생산관리자

| ID | 사용자 스토리 | 우선순위 |
|----|---------------|----------|
| US-PM-01 | 나는 생산관리자로서, 현재 진행 중인 절임 배치 목록과 각 배치의 남은 시간을 한눈에 보고 싶다. 그래야 생산 일정을 조율할 수 있다. | Must |
| US-PM-02 | 나는 생산관리자로서, 신규 절임 배치를 등록할 때 세척 배치와 자동으로 연계하고 싶다. 그래야 LOT 추적성을 유지할 수 있다. | Must |
| US-PM-03 | 나는 생산관리자로서, 계절별 절임 시간 기준을 조회하고 배치 등록 시 자동 적용하고 싶다. 그래야 품질 기준을 일관되게 지킬 수 있다. | Should |
| US-PM-04 | 나는 생산관리자로서, 일별/주별 소금 사용량 집계를 조회하고 싶다. 그래야 원가 관리를 할 수 있다. | Should |
| US-PM-05 | 나는 생산관리자로서, 절임 시간 초과 배치에 대한 자동 알림을 받고 싶다. 그래야 즉각 조치를 취할 수 있다. | Must |

### 2.2 품질관리자

| ID | 사용자 스토리 | 우선순위 |
|----|---------------|----------|
| US-QC-01 | 나는 품질관리자로서, 각 배치의 소금 농도 측정 이력을 시계열로 조회하고 싶다. 그래야 HACCP 기록 관리 의무를 이행할 수 있다. | Must |
| US-QC-02 | 나는 품질관리자로서, 소금 농도가 허용 범위를 벗어날 때 즉시 알림을 받고 싶다. 그래야 CCP 일탈에 신속히 대응할 수 있다. | Must |
| US-QC-03 | 나는 품질관리자로서, 절임 완료 배치에 대해 합격/불합격 판정을 내리고 그 근거를 기록하고 싶다. 그래야 HACCP 감사 대비가 가능하다. | Must |
| US-QC-04 | 나는 품질관리자로서, 불합격 배치의 시정조치 내용을 기록하고 싶다. HACCP 법적 요구사항이다. | Must |
| US-QC-05 | 나는 품질관리자로서, 기간별 품질 합격률 추이를 차트로 보고 싶다. 그래야 공정 개선 활동을 지원할 수 있다. | Could |

### 2.3 현장작업자

| ID | 사용자 스토리 | 우선순위 |
|----|---------------|----------|
| US-WK-01 | 나는 현장작업자로서, 내가 담당하는 배치의 현재 상태와 다음 행동을 명확히 알고 싶다. 그래야 실수 없이 작업할 수 있다. | Must |
| US-WK-02 | 나는 현장작업자로서, 소금 농도 측정값을 모바일에서도 간편하게 입력하고 싶다. 그래야 현장에서 바로 기록할 수 있다. | Should |
| US-WK-03 | 나는 현장작업자로서, 절임 완료 시점에 알림을 받고 싶다. 그래야 다음 공정으로 적시에 이관할 수 있다. | Must |
| US-WK-04 | 나는 현장작업자로서, 이상 발생 시 보고 절차를 시스템에서 안내받고 싶다. 그래야 빠짐없이 보고할 수 있다. | Should |

## 4. 화면 기능 명세

### 4.1 페이지 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│  PageHeader: 절임 공정 관리                               │
│  subtitle: HACCP CCP-2 절임 배치 현황 및 소금 농도 관리  │
├──────────────────────────────────────────────────────────┤
│  [요약 카드]  진행중 배치 | 오늘 완료 | 이상 발생 | 합격률│
├──────────────────────────────────────────────────────────┤
│  [탭 네비게이션]                                         │
│  [진행중 배치] | [소금 농도 이력] | [완료 판정] | [통계] │
├──────────────────────────────────────────────────────────┤
│  [탭 컨텐츠 영역]                                        │
└──────────────────────────────────────────────────────────┘
```

### 4.2 상단 요약 카드 (`SaltingSummaryCards`)

- 진행중 배치 수 (클릭 시 해당 배치 목록으로 필터)
- 오늘 완료 배치 수
- 이상 발생 건수 (미확인 알림 배지 표시)
- 당일 품질 합격률(%)

### 4.3 탭 1: 진행중 절임 배치 목록 (`ActiveSaltingBatchList`)

**기능 요구사항:**

- 현재 상태가 '진행중'인 배치를 카드 또는 테이블 형식으로 표시
- 남은 시간 카운트다운 실시간 표시 (목표완료시간 - 현재시간)
  - 30분 이하: 주황색 경고 표시
  - 시간 초과: 빨간색 + 알림 뱃지
- 각 배치 행에서 즉시 접근 가능한 액션 버튼:
  - [소금 농도 측정 입력]
  - [온도 기록]
  - [완료 처리]
  - [이상 신고]
- 컬럼 구성: 배치번호 | LOT | 원재료 | 투입량 | 목표농도 | 최근측정농도 | 시작시간 | 남은시간 | 담당자 | 상태 | 액션

**화면 상태별 표시:**

| 상태 | 배경색 | 아이콘 | 설명 |
|------|--------|--------|------|
| 대기 | 회색 | 시계 | 아직 시작 전 |
| 진행중 | 초록 | 재생 | 정상 진행 중 |
| 완료임박 | 주황 | 알람 | 30분 이내 완료 예정 |
| 시간초과 | 빨강 | 경고 | 계획 시간 초과 |
| 이상 | 빨강 (진) | X | 이상 발생 |

### 4.4 탭 2: 소금 농도 측정 이력 (`SaltConcentrationHistory`)

**기능 요구사항:**

- 배치 선택 드롭다운 (진행중 배치 우선 표시)
- 선택된 배치의 농도 측정값을 시계열 라인 차트로 시각화
  - X축: 측정 시간
  - Y축: 농도(%)
  - 허용 범위(목표 ±1%) 상/하한선 점선으로 표시
  - CCP 한계기준(10%/15%) 실선으로 표시
- 측정 이력 테이블 (시간 역순 정렬):
  - 측정 구분 | 농도값 | 허용범위 이탈 여부 | 측정자 | 측정시간 | 비고
- 신규 측정값 입력 버튼 ([+ 농도 측정 입력])

**신규 측정 입력 모달:**

```
┌──────────────────────────────────┐
│  소금 농도 측정 입력              │
├──────────────────────────────────┤
│  배치번호: SB-20260513-001 (고정) │
│  측정 구분: [초기/중간/최종] 선택 │
│  측정 농도(%): [____] 숫자 입력   │
│  측정 시간: [자동입력, 수정 가능] │
│  비고: [텍스트 입력]              │
├──────────────────────────────────┤
│  허용범위: 12.0% ± 1.0%          │
│  현재 입력값 범위 이탈 시 경고    │
├──────────────────────────────────┤
│       [취소]     [저장]           │
└──────────────────────────────────┘
```

### 4.5 탭 3: 완료 판정 처리 (`SaltingCompletionJudgment`)

**기능 요구사항:**

- 완료 처리 대기 중인 배치 목록 표시 (상태: 진행중 + 계획 시간 경과)
- 각 배치의 전체 측정 이력 요약 표시
- 합격/불합격/보류 판정 버튼
  - 불합격 선택 시: 시정조치 내용 입력 필드 노출 (필수 입력)
  - 합격 판정 시: 탈수 공정으로 배치 자동 이관 처리
- HACCP 기록 확정: 판정 완료 후 CCP 기록 잠금(수정 불가 처리)

**판정 기준 체크리스트 (자동 계산, 참고용):**

- [ ] 최종 소금 농도가 허용 범위 내인가?
- [ ] 실제 절임 시간이 최소 기준 이상인가?
- [ ] 중간 점검 측정이 완료되었는가?
- [ ] 이상 발생 이력이 없는가?

### 4.6 이상 발생 알림 패널 (`SaltingAlertPanel`)

**알림 유형별 처리:**

| 알림 유형 | 트리거 조건 | 심각도 | 에스컬레이션 대상 |
|-----------|-------------|--------|------------------|
| 농도이탈 | 측정값이 목표 ±1% 초과 | 경고 | 품질관리자 |
| 농도 CCP 한계 이탈 | 10% 미만 또는 15% 초과 | 위험 | 품질관리자 + 생산관리자 |
| 시간초과 | 계획 완료 시간 + 30분 경과 | 경고 | 생산관리자 |
| 최대 시간 도달 | 절임 시간 8시간 초과 | 위험 | 생산관리자 + 품질관리자 |
| 온도 이탈 | 절임 온도 15℃ 미만 또는 20℃ 초과 | 경고 | 현장관리자 |
| 완료 임박 | 목표 완료 시간 30분 전 | 정보 | 담당 작업자 |

**알림 표시 방식:**

- 페이지 우측 상단 알림 뱃지 (미확인 건수)
- 알림 패널 슬라이드아웃: 알림 목록, 확인 처리 버튼
- 위험 등급 알림: 화면 상단 빨간색 배너 표시 (확인 전까지 유지)

### 4.7 탭 4: 소금 사용량 집계 및 통계 (`SaltingStatistics`)

- 기간 선택 필터 (오늘/이번 주/이번 달/사용자 지정)
- 소금 사용량 집계 차트 (원재료 종류별 막대 차트)

- CSV 내보내기 버튼

## 6. 비즈니스 규칙

### 6.1 소금 농도 허용 범위

```
목표 농도 설정값을 기준으로:
- 정상 범위: 목표 농도 ± 1.0%
- 경고 범위: 목표 농도 ± 1.0% 초과 ~ ± 2.0% 이하
- HACCP CCP 한계 이탈: 10% 미만 또는 15% 초과 (배추 기준)
- 원재료별 기준:
  - 배추: 목표 12%, 허용 11~13%
  - 무: 목표 10%, 허용 9~11%
  - 열무: 목표 8%, 허용 7~9%
```

### 6.2 절임 시간 기준

```
원재료: 배추 기준

최소 절임 시간: 4시간 (이 미만은 불합격 처리 불가, 경고 필수)
최대 절임 시간: 8시간 (초과 시 품질 저하 위험, 위험 알림 발생)
권장 절임 시간: 5시간 (계절 보정 전 기준)

계절별 보정 로직:
  - 봄(3~5월): 기준 시간 × 1.0 (보정 없음)
  - 여름(6~8월): 기준 시간 × 0.85 (기온 상승으로 단축)
  - 가을(9~11월): 기준 시간 × 1.0 (보정 없음)
  - 겨울(12~2월): 기준 시간 × 1.3 (저온으로 연장)

보정 시간 범위는 최소/최대 기준 이내로 제한:
  예) 여름 기준: 권장 5h × 0.85 = 4.25h (최소 4h 이상이므로 적용)
  예) 겨울 기준: 권장 5h × 1.3 = 6.5h (최대 8h 이내이므로 적용)
```

### 6.3 이상 발생 시 에스컬레이션 절차

```
Step 1: 자동 알림 생성
  - 시스템이 이상 조건 감지 시 즉시 알림 레코드 생성
  - 담당 작업자에게 화면 알림 및 (옵션) 모바일 푸시 발송

Step 2: 1차 확인 (담당 작업자, 15분 이내)
  - 작업자가 알림 확인 및 현장 상황 점검
  - 즉각 해결 가능한 경우: 시정조치 기록 후 종결
  - 해결 불가한 경우: Step 3로 에스컬레이션

Step 3: 2차 에스컬레이션 (생산관리자/품질관리자, 30분 이내)
  - 위험 등급 알림 또는 15분 내 미확인 시 자동 에스컬레이션
  - 관리자가 배치 처리 방향 결정 (계속/중단/조건부 합격)

Step 4: CCP 한계 이탈 처리 (위험 등급)
  - HACCP 시정조치 기록 필수 (시스템에서 강제)
  - 해당 배치 격리 또는 폐기 처리
  - 원인 분석 및 재발방지 대책 문서화
  - 위생관리자에게 즉시 보고 (HACCP 법적 요구)

미확인 경보 자동 에스컬레이션 타임라인:
  경고 등급: 30분 미확인 → 담당 관리자에게 알림
  위험 등급: 15분 미확인 → 즉시 생산관리자 + 품질관리자 알림
```

### 6.4 배치 상태 전환 규칙

```
대기 → 진행중: 시작시간 기록 후 수동 전환 (또는 시작 시간 도달 시 자동)
진행중 → 완료: 완료 판정(합격/불합격) 처리 시 전환
진행중 → 이상: 이상 신고 접수 시 수동 전환
이상 → 진행중: 이상 해소 확인 후 관리자 수동 전환 가능
완료 → (수정 불가): 합격 판정 완료 후 HACCP 기록 잠금
불합격 → 재작업: 별도 재작업 배치 생성 (원 배치는 불합격으로 종결)
```

### 6.5 데이터 무결성 규칙

- 절임 배치 번호는 `SB-YYYYMMDD-NNN` 형식으로 자동 채번
- 세척 배치와 1:1 연계 필수 (중복 연계 불가)
- 최종 판정 전 최소 1회 소금 농도 측정 기록 필수
- HACCP 기록(농도측정 이력, 판정 결과)은 등록 후 수정/삭제 불가 (감사 추적 보장)
